## 📖Overview

This project is the fundation of AI enginner training , its a simple Rule-based AI chabot built with python.

The bot take user text input , compare it with fixed set of knowns phrases and returns a response.
if nothing matches , it falls back to default message.
---

## ✨features :
-Greets the user with friendly responses
-Uses a dictionary for responses
-Cleans user input using '.lower()' and '.strip()'
-Exits when the user types 'bye' or 'exit'
-Handle unknown inputs with fallback response
---

## 🎯 Project goal :
create a simple rule-based chatbot that :
    -handles greetings and exit commands.
    -responds using predefined rules.
    -Runs in a continuous loop until the user chooses to exit.
---
## ⚙️ How it works :
 1. Input 
 the chatbot waits for the user to enter a message 
 clean the input with '.lower()' and '.strip()'
 2. Process
the chatbot stores the responses in Python dictionary
each user input (key) is linked to a response (value)
if the message exist , it returns the correct response
if not , it display a default message
3. output
the chatbot prints the response on the screen and waits for another message.
